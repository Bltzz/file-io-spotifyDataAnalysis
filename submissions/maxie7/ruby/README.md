# DataLogger

This is a solution for Hacktoberfest 2023 in Ruby (v 3.2.1)

## Installation

0. You should have Ruby installed in your system

1. Get to the folder `submissions/maxie7/ruby/` in the terminal

2. Run this:

```ruby
ruby spotify_data_logger.rb
```
